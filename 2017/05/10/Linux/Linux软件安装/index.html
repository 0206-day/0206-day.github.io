Linux软件安装

规划目录

  1. 首先先清空 /opt 目录
    rm -rf /opt/*  --》 没有进入到opt目录下
rm -rf ./*     --》 已经进入到opt目录下
  2. 创建目录
    mkdir datas    --》 存放文件数据的
mkdir modules  --》 存放安装软件的
mkdir tools    --》 存放开发工具的
mkdir software --》 存放安装包的
mkdir workspace --》 存放项目源码的

安装JDK

1.查看java版本：java -version
2.查看系统自带JDK
    rpm -qa | grep jdk
    如果有则全部卸载
    rpm -e --nodeps java-XXX.86_64 
3.从官网下载所需要的JDK
4.上传至Linux /opt/software
    rz jdk-7u67-linux-x64.tar.gz
5.解压JDK
    tar -xzf /opt/software/jdk-7u67-linux-x64.tar.gz -C /opt/modules/
6.配置环境变量
    vi /etc/profile
    在文件的第二行输入
    #JAVA_HOME
    export JAVA_HOME=/opt/modules/jdk1.8.0_192
    export PATH=$PATH:$JAVA_HOME/bin
7.生效配置文件
    source /etc/profile
    退出用户 exit
    重新测试 java -version

测试环境变量是否配置成功 java

安装Eclipse

1.官网下载Eclipse安装包
2.上传至Linux /opt/software
    rz eclipse-jee-kepler-SR1-linux-gtk-x86_64.tar.gz
3.解压Eclipse
    tar -xzf /opt/software/eclipse-jee-XXX-x86_64.tar.gz -C /opt/tools/
4.运行Eclipse
    方式一：通过图标点击，创建连接链接文件启动
    方式二：命令 /opt/tools/eclipse/eclipse
    方式三：配置环境变量
        vi /etc/profile
        在空白处填写
            #ECLIPSE
            export ECLIPSE_HOME=/opt/tools/eclipse
            export PATH=$PATH:$ECLIPSE_HOME
        生效配置文件
            source /etc/profile

安装Tomcat

1.官网下载Tomcat安装包
2.上传至Linux /opt/software
    rz apache-tomcat-7.0.85.tar.gz
3.解压Tomcat
    tar -xzf /opt/software/apache-tomcat-7.0.85.tar.gz -C /opt/modules/
4.启动Tomcat
    /opt/modules/tomcat7/bin/starup.sh
5.查看启动是否成功
    方式一： jps 查看进程   
        3238 Bootstrap
        3277 Jps
    方式二： 通过web浏览器。输入ip地址和端口号访问

http://txx.linux:8080/
注意：
1.不要用windows自带的浏览器
2.如果访问不到，请检查linux防火墙是否关闭
6.安装服务
1）拷贝catalina.sh 到 /etc/init.d/tomcat
cp -p /opt/modules/tomcat7/bin/catalina.sh /etc/init.d/tomcat
2）编辑 vi /etc/init.d/tomcat 文件，在第二行加入以下内容
# chkconfig: 112 63 37
# description: tomcat server init script
# Source Function Library
. /etc/init.d/functions 

        JAVA_HOME=/opt/modules/jdk1.7.0_67
        CATALINA_HOME=/opt/modules/tomcat7
    3）保存文件后修改成以下权限
        chmod 755 /etc/init.d/tomcat
    4）添加tomcat到服务中
        chkconfig --add tomcat  --》 添加到服务中
        chkconfig tomcat on     --》 每次开机自启
    5）服务
        启动服务：service tomcat start
        停止服务：service tomcat stop
        重启服务：service tomcat restart
