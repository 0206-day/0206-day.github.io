Linux常用命令
发表于2017-05-06|更新于2017-05-06|Linux
|阅读量:

1.重启:

 【init 6】

【shutdown -r now】

【reboot】

2.关机 :

【init 0】

【shutdown -h now】

3、查看网络ip地址:

【ifconfig】

4.重启网络配置

【service network restart】

5.打开文件

【gedit +文件地址】

6、时间

【date】——》查看时间

【date -s ‘2019-1-15 14:05:00’】——》修改时间

7、主机名

【hostname】——》查看主机名

【hostname aaabbb】——》临时修改主机名

【vim /etc/sysconfig/network】——》永久修改主机名

8、[root@JYHadoop02 opt]#

root：代表当前登录的用户

JYHadoop02：主机名

opt：当前打开的文件名

#：root用户

$：普通用户

9、清屏

clear

10、显示当前所在目录

pwd

11、切换用户

切换到test用户：su test

12、命令名称：whoami

​ 语法： whoami

​ 功能： 查看当前操作用户

13、 命令名称：whatis

​ 语法： whatis [命令名称]

​ 功能： 查看帮助命令

14、命令名称：whereis

​ 语法： whereis [文件名称]

​ 功能： 查看文件位置

15、命令名称：which

​ 语法： which

​ 功能： 查看可执行文件的位置

16、查看当前操作系统发行信息：cat /etc/centos-release

17、命令名称：find

语法： find 文件名

功能：查看文件位置

find 查找文件会消耗cpu巨大资源，不轻易使用
