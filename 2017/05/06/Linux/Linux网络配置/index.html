Linux网络配置

配置静态网络

配置网卡

1、在VM中–》编辑 –》 虚拟网络编辑器 –》VMnet8 NAT模式中 192.168.160.0
2、在Linux系统中找到网络连接图标右击 –》编辑连接
1）勾选自动连接
2）选择IPV4设置，方法改为手动，添加ip地址
网络三要素配置如下
IP地址：将虚拟机网络的前三组照抄，最后一组改为11255之间任意数字
如：192.168.160. (11255之间的数字)
子网掩码：所有都一样：255.255.255.0
网关IP：在虚拟机中网关IP是IP地址最后一组改为2如：192.168.160.2
DNS：114.114.114.114
3)配置Linux中的hosts文件
gedit /etc/hosts 最后一行写入
192.168.195.131 CC.linux(主机名)
4)重启网络: service network restart
5)重启计算机 init 6
3、在windows中修改:C:\Windows\System32\drivers\etc\hosts 文件
配置内容:
192.168.195.131 JYHadoop.linux(主机名)
4、通过配置文件修改IP地址：
路径/etc/sysconfig/network-scripts/ifcfg-eth0

配置防火墙

CentOS-6

1. 临时使用防火墙
    service iptables stop   --》 临时停止防火墙
    service iptables start  --》 临时启动防火墙
    service iptables status --》 查看防火墙的状态
2. 永久操作防火墙
    chkconfig iptables off  --》 永久关闭防火墙
    chkconfig iptables on   --》 永久开启防火墙

注意：关闭顺序，首先是临时关闭，最后才是永久关闭
3. 开放端口 (不需要关闭防火墙，外部可以正常访问)
1）vi /etc/sysconfig/iptables
2）在COMMIT之前添加
-A INPUT -m state –state NEW -m tcp -p tcp –dport 8080 -j ACCEPT
并保存退出
3）重启防火墙
/etc/init.d/iptables restart
