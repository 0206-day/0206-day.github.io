Linux文件压缩和打包

Linux常用压缩和解压缩格式

​ tar

​ tar.gz(tgz)

​ tar.bz2

命令名称：tar

​ 打包命令：tar [-选项] 名称 目标

​ 解包命令：tar [-选项] 目标 -C 指定位置

​ 选项：-c：打包 -x：解包 -v：显示打包信息 -t：查看包信息

​ -z：gzip -j：bzip2 -f：使用文档名(必填)

​ 打包示例：

​ –> 打包生成file.tar 包(储存在默认位置)

​ tar -cvf file.tar init.txt passwd.txt test.java

​ –> 打包生成file.tar包(指定到/root/位置)

​ tar -cvf /root/file.tar init.txt passwd.txt test.java

​ –> 将/opt/目录下所有文件打包生成2018.tar包

​ tar -cvf 2018.tar /opt/*

​ 解压示例：

​ –> 解压文件(默认位置)

​ tar -xvf file.tar

​ –> 解压文件(到指定位置 -C)

​ tar -xvf file.tar -C /opt/aaa/

​ 查看示例：

​ tar -tf 2018.tar

.tar.gz(tgz)

​ 压缩：

​ tar -czvf etc20181207.tar.gz /etc/

​ –> -z：gzip压缩格式

​ 解压：

​ tar -xzvf etc20181207.tar.gz -C tom001/

.tar.bz2

​ Linux下压缩比率较tgz大，即压缩后占用更小的空间，使得压缩包看起来更小。

​ 但同时在压缩，解压的过程却是非常耗费CPU和时间

​ 压缩：

​ tar -cjvf etc20181207.tar.bz2 /etc/

​ –> j：bzip2压缩格式

​ 解压：

​ tar -xjvf etc20181207.tar.bz2 -C rh/

注意事项：

​ 1.使用压缩格式进行压缩的文件，必须用同样的格式进行解压

​ 2.相互冲突的选项不能一起使用

​ 3.压缩或解压的选项顺序：f 一般在最后一个，v 一般在f的前面

​ 4.用什么样压缩方式压缩的，就用什么解压方式解压

压缩格式拆解

​ 注意：只能压缩文件不能压缩目录,

​ 压缩完成后源文件会消失，生成一个以源文件名为名称的压缩文件，

​ 解压时，压缩文件会丢失。

​ gzip

​ 后缀：.gz

​ 压缩：gzip file

​ 解压：gunzip file.gz

​ bzip2

​ 后缀：.bz2

​ 压缩：bzip2 file

​ 解压：bunzip2 file.bz2
