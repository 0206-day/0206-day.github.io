Linux权限管理

注意：默认外来文件没有执行权限

Linux命令行中的知识

在Linux命令中第一个字符表示：
当为【d】则是目录；
当为【-】则是文件；
当为【l】则表示为连接文档(link file)；
当为【b】则表示为装置文件里面的可供存储的接口设备(可随机存储装置)；
当为【c】则表示为装置文件里面的串行端口设备，例如键盘、鼠标(一次性读 取装置)：
在Linux中后9位字符每3位代表一组：
第一组代表所属者权限：所属者（u）
第二组代表所属组用户：所属组用户（g）
第三组代表其他用户：其他用户（o）

修改权限的方法

第一种方法：通过权限字符来修改：用于对权限需要严格控制的文件或目录
用户表示：所属者（u）、所属组用户（g）、其他用户（o）、所有用户（a）
权限表示：读（r）、写（w）、执行（x）
操作实例
去除所有者读的权限：【chmod u-r 文件名或目录名】
同组者增加写的权限：【chmod g+w 文件名或目录名】
所有人都可执行权限：【chmod u+x,g+x,o+x 文件名或目录名】简写：【chmod a+x 文件名或目录名】-R迭代修改，在修改目录时，目录内的所有文件及目录一起被修改：chomd -R o-rx rh/
第二种方法：通过数字修改
（1）用户表示：三位二进制
（2）文件操作：读（4）、写（2）、执行（1）
2、文件所属者、所属组权限
（1）更改目录所属的组
将a.txt文件放到student组中：【chgrp student a.txt】
（2）更改目录所属者和所属组
将a.txt文件所属者、所属组为tom：【chown tom:tom a.txt】

sudo权限命令

1、执行越权命令
执行越权命令：【sudo 要执行的命令】
查看可执行的越权命令：【sudo -l】
在切换至tom用户执行重启命令：【sudo /sbin/shutdown -r now】
在切换至tom用户执行关机命令：【sudo /sbin/shutdown -h now】
执行越权命令的前提是赋予了越权命令
2、赋予越权命令
切换至root用户输入【visudo】命令回车在98行处添加标红的字给tom用户赋予越权命令重启、关机
