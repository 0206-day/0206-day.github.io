Linux用户和用户组管理

1、用户

​ 1）UID 0 代表root用户

​ 2）普通用户UID从500开始

​ 3）用户和用户信息通常存储在/etc/passwd文件中

2、用户组

​ 1）GID 0 代表root用户

​ 2）普通用户GID从500开始

​ 3）同一个组中的所有用户能共享属于这个组的文件

3、新增用户及初始化用户密码

​ 新建一个test用户：【useradd test】

​ 初始化test用户的密码为1234566：【passwd 用户名】

4、删除用户

​ 删除jack用户：【userdl 用户名】

​ 删除jack用户及其记录(目录)：【userdl -r 用户名】

5、新增用户组

​ 新增一个student用户组：【groupadd 组名】

6、删除用户组

​ 删除student用户组：【groupdel 组名】

7、将某个用户添加到某个组

​ 命令：【gpasswd [选项] [用户名] [组名]】

​ 选项：-a是添加 -d是删除

​ 将jack用户添加到student组中：【gpasswd -a jack student】

​ 将jack用户从student组中删除：【gpasswd -d jack student】

8、文件所属者、所属组权限

     （1）更改目录所属的组

​ 将a.txt文件放到student组中：【chgrp student a.txt】

     （2）更改目录所属者和所属组

​ 将a.txt文件所属者、所属组为tom：【chown tom:tom a.txt】

9、退出当前用户

​ 【exit】
