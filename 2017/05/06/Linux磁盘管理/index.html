Linux磁盘管理

1、磁盘
磁盘分区类型：
主分区：存放系统，系统配置文件工具等
扩展分区： 扩展分区，顾名思义，用来做系统扩展使用， 并不能储存任何数据
逻辑分区： 实际储存数据及安装软件等需要时用的磁盘
主分区+扩展分区 <= 4
管理命令：fdisk
使用方法：
1、查看当前磁盘及分区信息
fdisk -l
2、对已存在磁盘设备分区
fdisk DEVICE_FILE_PATH
查看Linux系统磁盘空间：df -h
查看挂载： mount -l
新建主分区步骤：
查看磁盘信息： fdisk -l
管理磁盘分区： fdisk /dev/sdb
创建分区： n
选择分区类型： p
分区号： 1
磁头开始位置： 回车
结束位置（大小）：+10G
查看分区信息： p
保存退出： w
格式化分区： mkfs.ext4 /dev/sdb1
挂载分区： mount /dev/sdb1 /media
查看挂载： df -h
永久挂载：
vi /etc/fstab
/dev/sdb1 /media ext4 default 0 0
新建扩展分区步骤：
查看磁盘分区信息： fdisk -l
管理磁盘分区： fdisk /dev/sdb
创建分区： n
选择分区类型： e
分区号： 2
磁头开始位置： 回车
结束位置（大小）：+5G
查看分区信息： p
在扩展分区上创建逻辑分区步骤：
创建分区： n
选择分区类型： l
磁头开始位置： 回车
结束位置（大小）：回车 全部大小
查看分区信息： p
保存退出： w
重启
格式化分区： mkfs.ext4 /dev/sdb5
挂载分区： mount /dev/sdb5 /media
查看挂载： df -h
永久挂载：
vi /etc/fstab
/dev/sdb5 /media ext4 defaults 0 0
