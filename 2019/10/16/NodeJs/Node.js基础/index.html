Node.js基础认识

发表于2019-10-16|更新于2019-10-16|Java
|阅读量:

Node.js基础

简介

  1. Node.js 是一个基于 Chrome V8 引擎 的 JavaScript 运行时。(官网解释)
    1. Node.js不是一门语言
    2. 不是库、不是框架
    3. 以前只有浏览器可以解析和运行JS，有了Node.js以后JS可以脱离浏览器运行
    4. Node.js构建与Chrome的V8引擎之上
      1. 代码只是具有特定格式的字符串，引擎可以认识它解析它，并运行它
      2. Node.js 的作者把Google Chrome中的V8引擎移植了出来，开发了一个独立的JavaScript运行环境
  2. 浏览器中的JavaScript由以下三部分组成
    1. EcmaScript
      1. if
      2. var
      3. function
      4. Object
      5. Array
    2. BOM(Browser Object Model)浏览器对象模型
    3. DOM(Document Object Model)文档对象模型
  3. Node.js 中的JavaScript
    1. 没有BOM、DOM
    2. 只有EcmaScript，因为服务端不操作HTML页面
    3. Node.js为JS提供了一些操作服务器的API
      1. 文件读写
      2. 网络服务的构建
      3. 数据网络通信
      4. http 服务器等等……
  4. Node.js 使用了事件驱动，非阻塞I/O模型，这使得它轻量级且高效（官网解释）
    1. 特性
      1. event-driven 事件驱动
      2. non-blocking I/O model 非阻塞I/O模型（异步）
  5. Node.js 的包生态系统npm是世界上最大的开源库生态系统（官网解释）
    1. npm包管理工具是基于Node.js开发的

作用

node.js 目前能做两件事

  • Web 服务器后台
  • 命令行工具
    • npm（node开发）
      • node.js包管理工具（Node Package mangager，npm），前端主流技术基本都会用npm来管理依赖。类似Maven
    • git （c语言开发）
    • hexo（node开发）
    • 对于前端而言，使用node最多的就是它的命令行工具
      • webpack
      • gulp
      • npm

安装

Windows

1、Node.js下载地址：http://nodejs.cn/download/
2、公司开发项目，建议稳定版。以学习目的的，建议最新版。
3、安装
.msi：是安装包。双击运行，一直下一步即可安装
.zip：是压缩包，解压后配置以下环境变量即可使用

4、打开命令窗口输入 node –version 或 node -v 查看是否安装成功
5、安装Node.js淘宝镜像加速器(cnpm)

6、配置npm下载依赖位置
npm在现在第三方依赖时会先将依赖下载到npm的缓存文件处，最后在存在本地仓库

配置完以后可以用 npm config ls 查看配置结果。.npmrc文件也会变化，该文件在C盘的用户目录下

npm

npm(Node Package Mangager)是Node.js的包管理工具，可用于管理前端依赖包
npm i：该命令其实就是 npm install 的简写但又有些不同。此命令会帮助检测与当前node最匹配的版本号，并匹配出相互依赖的npm包应该升级的版本号。使用此命令安装的依赖，使用npm nuinstall无法删除，必须使用npm uninstall i才能删除。

npm  i  xxx -D：就是 npm  i  xxxx –save-dev  是把依赖写入进devDependencies对象里面，devDependencies是开发环境下的依赖，这里是开发环境下的依赖，上线后非必需，比如：webpack，gulp等压缩打包工具
npm  i  xxx -S：就是 npm  i  xxxx –save       是把依赖写入进dependencies对象里面    是生产环境下的，是刚需的 如UI框架  文件字体，dependencies 是生产环境下的依赖，项目刚需的依赖在这里，比如UI框架，字体文件等线上必需的东西
总结：-S 是刚须的，是生产环境下的，如UI字体库
           -D是开发环境下的，项目上线后，不一定需要，如webpack, Gulp等。
